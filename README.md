 🍕 Pizza Delivery - Proiect POO (Tema 1)

 Descriere proiect:
-> Am incercat sa simulez modul de operare a unei pizzerii: alegerea pizzei, introducerea datelor de contact si a unei adrese de livrare, ca la final sa fie afisata o chitanta cu toate informatiile.

 Clasele folosite:
-> Pizza
  reprezinta produsul in sine.
  atribute: nume, nr. de ingrediente, lista de ingrediente si pretul.

-> Adresa
  gestioneaza locatia de livrare.
  atribute: oras, strada, numar si detalii suplimentare.

-> Client
  clasa centrala care leaga logica 'aplicatiei'.
  atribute: nume, nr. de telefon si adresa de livrare (clasa Adresa).
